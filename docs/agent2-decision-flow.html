layer 2 — calculation only, no gate
Expected move
BS move = spot × ATM IV × √(DTE ÷ 365)  ·  1.4 SD boundary = BS move × 1.4
Cross-check: ATM straddle LTP (CE + PE)  ·  flag divergence if methods differ > 15%
↓
layer 3 — calculation only, no gate
Strike selection
Credit spreads — short strike at 1.4 SD boundary · delta ≤ 0.20 · round to nearest 50 · spread width: DTE ≤ 4 → min, DTE > 4 → max (both from user profile)
Debit spreads — long = ATM · short = ATM + 200 pts rounded to nearest 50
↓
layer 4 — gate validation round 1 (G1 · G2 · G3)
PoP, max loss, and spread width checks
HARD G1 PoP — credit: seller PoP = (1 − short leg pop) × 100 ≥ 80%  ·  debit: long leg pop × 100 ≥ 35%
INFO G2 max loss — max loss per lot calculated for sizing · never rejects · always passes
HARD G3 PoP–PoPP gap — |short leg pop − long leg pop| × 100 ≤ 15% · credit spreads only, auto-pass for debit
→
REJECTED
G1 or G3 fails
↓
layer 5 — position sizing
Lot calculation
Real expected loss / lot = max loss / lot × 0.50 (Agent 3 exits at T2 — not full max loss)
Max loss allowed = capital × max_loss_pct (user profile default 1.5%)
Lots = floor(max loss allowed ÷ real expected loss / lot)
↓
layer 4 — gate G4 (evaluated after sizing)
Minimum return check
HARD G4 RoC — RoC ≥ 0.5% × (DTE ÷ 7)  ·  DTE 3 → min 0.21%  ·  DTE 7 → min 0.50%
→
REJECTED
G4 fails
