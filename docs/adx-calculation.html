ADX Calculation — Step-by-Step Walkthrough
Average Directional Index · Wilder smoothing · 6-bar Nifty example with N = 3 period · Agent 1 confidence modifier
What ADX measures
Purpose
ADX measures trend strength, not direction. A value above 30 means a strong trend exists — bullish or bearish. Below 20 means sideways or choppy. Agent 1 uses it to adjust confidence: strong trend → boost confidence; weak/sideways → reduce it.
In Agent 1
Source: 200-day Nifty OHLC from Upstox historical API.
Library: TA4J ADXIndicator(series, 14) — period N = 14 in production.
Applied as: confidence × adxModifier
Requires ≥ 28 bars minimum (2 × N). Null → modifier = 1.00.

Step 3 — Wilder Smoothing (N = 3)
STEP 3 Smooth TR, +DM, −DM using Wilder's method
Wilder's Smoothing formula
Initialisation: SmoothedN[0] = sum of first N raw values
Subsequent bars: SmoothedN[i] = SmoothedN[i-1] − SmoothedN[i-1]/N + today
This is equivalent to EMA with α = 1/N. With N = 3, α = 0.333.
Smooth TR (raw values: 170, 230, 220, 210, 190, 260)
SmTR[init] = bars 1+2+3170 + 230 + 220 = 620(covers bars 1–3)
SmTR[bar 4] = 620 − 620/3 + 210620 − 206.67 + 210 = 623.33
SmTR[bar 5] = 623.33 − 623.33/3 + 190623.33 − 207.78 + 190 = 605.56
SmTR[bar 6] = 605.56 − 605.56/3 + 260605.56 − 201.85 + 260 = 663.70

Step 5 — DX and ADX
STEP 5 DX = directional divergence ratio · ADX = Wilder smooth of DX
DX = |+DI − −DI| / |+DI + −DI| × 100 ADX[init]  = average of first N DX values ADX[i]     = (ADX[i-1] × (N-1) + DX[i]) / N   same as Wilder smoothing

* In this toy 6-bar example, DX happens to be 100 on all available bars because every day was strongly bullish with no −DM. In production with N = 14 and 200+ bars, mixed trading days create DX values spread across 0–100, and ADX typically settles in the 15–60 range depending on trend strength. The 47.8 figure used in the interactive widget was derived from a representative Nifty data set with partial bearish days included for illustration.

Example: If Agent 1 base confidence = 0.60 (MEDIUM) and VIX is NORMAL (×1.00) and ADX = 47.8 > 30 (×1.15):
Final confidence = 0.60 × 1.00 × 1.15 = 0.69 → still MEDIUM (just under the HIGH threshold of 0.70).
Same scenario with ADX < 20 → 0.60 × 1.00 × 0.80 = 0.48 → MEDIUM (but lower, closer to LOW boundary).
